# 📊 Work Management Assistant (WMA)

A premium, high-integrity Resource Planning System designed for team leaders and managers. WMA provides a real-time command center for tracking workforce utilization, task smoothing, and strategic insights.

## 🚀 Key Features

- **Intelligence-Driven Resource Planning**: Automatically smooths task hours over long-term durations to prevent weekly burnout.
- **Executive Command Center**: Real-time metrics dashboard tracking Team Capacity, Utilization %, and Monthly Pipeline.
- **Manager Insights Engine**: Dynamic notification center that flags burnout risks, missed deadlines, and identifying top available resources.
- **Leader Overrides**: Full manual control over team statuses (Force Available, Busy, Away) and private manager notes.
- **Instant Operations**: Zero-latency management with no confirmation pop-ups for lightning-fast task reassignment and deletion.
- **Global Search Matrix**: Instantly filter your workforce by name, skill, or manager-defined traits.

## 🛠️ Technology Stack
- **Core**: Vanilla JavaScript (ES6+), HTML5, CSS3
- **Design**: Premium Glassmorphism, CSS Grid & Flexbox
- **Icons**: Phosphor Icons
- **Typography**: Google Fonts (Outfit, Inter)
- **Data**: Local-first with robust Backup/Restore functionality.

## 📦 How to Use
1. **Add Team Members**: Define their weekly capacity, skills, and unique traits.
2. **Assign Tasks**: Provide descriptions, estimated hours, and due dates.
3. **Optimize**: Use the "Suggest" engine to find the best candidate based on skill-match and current weighted workload.
4. **Monitor**: Watch the Command Center metrics at the top to ensure your team's health and project velocity.
